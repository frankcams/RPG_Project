Hero's Light.
Hero’s Light là một game RPG đánh theo lượt (Turn-based RPG) với cốt truyện sâu sắc, hệ thống chiến đấu yêu cầu tính toán chiến thuật và sắp xếp đội hình nhân vật qua từng lượt đánh.
